# Design Tokens Reference

Quick reference guide for all design tokens used in the Verifyr design system.

## Colors

### Primary Palette
- **Primary Blue**: `#3B82F6` - Main brand color
- **Dark Blue**: `#1E40AF` - Hover states, emphasis
- **Light Blue**: `#DBEAFE` - Background accents
- **Accent**: `#60A5FA` - Secondary accent

### Neutral Palette
- **Dark**: `#0F172A` - Primary text
- **Gray**: `#64748B` - Secondary text
- **Light Gray**: `#F1F5F9` - Backgrounds
- **White**: `#FFFFFF` - Base background

### Semantic Colors
- **Error**: `#EF4444`
- **Success**: `#065F46`
- **Success Background**: `#D1FAE5`
- **Error Background**: `#FEE2E2`

## Border Radius

- **Small**: 12px (buttons, small elements)
- **Medium**: 16px (cards, inputs)
- **Large**: 24px (sections, large cards)
- **Pill**: 50px (pill-shaped elements)
- **Full**: 50% (circles)
